Quiz App

This is a simple quiz application built using React.
I made this project while learning React to understand state management and component structure.

---

Features

* User can select a category (Cricket, Cinema, DSA)
* Questions are shown based on selected category
* Shows correct/wrong answer immediately
* Keeps track of score
* Option to restart quiz
* Option to go back and choose another category

---

Tech Used

* React
* JavaScript
* CSS

---

How to run

Clone the repo:

```
git clone https://github.com/your-username/quiz-app.git
```

Go to project folder:

```
cd quiz-app
```

Install dependencies:

```
npm install
```

Run the app:

```
npm run dev
```

Open in browser:

```
http://localhost:5173
```
